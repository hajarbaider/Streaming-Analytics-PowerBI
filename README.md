
# 📊 Streaming Platforms Analytics (Power BI Project)

## 📌 Project Overview
This project analyzes and compares the content catalogs of Netflix and Amazon Prime Video using Power BI.

The goal is to identify:
- Content distribution (Movies vs Series)
- Genre trends
- Duration analysis
- Age classification
- IMDb ratings
- Awards analysis

## 🛠 Tools Used
- Power BI Desktop
- DAX
- Data Modeling (Fact & Dimension Tables)
- Data Cleaning & Transformation

## 📂 Datasets
- Amazon Movies and TV Shows Dataset (9687 rows)
- Netflix Extended Dataset (15,480 rows)

## 📈 Key Insights
- Amazon catalog is dominated by Movies (81%)
- Netflix shows strong growth in 2020 releases
- Most Netflix content duration: 1-2 hours
- Adult-rated content dominates Netflix
